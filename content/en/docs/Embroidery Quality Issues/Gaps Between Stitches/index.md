# Gaps Between Stitches

Gaps between stitches are a common embroidery quality issue. They usually appear as visible spaces between satin stitches, letters, or design elements after embroidery.

This issue is often related to **digitizing settings** rather than the embroidery machine itself.

<img src="Gaps Between Stitches.jpg" alt="Gaps Between Stitches" style="width:100%; max-width:800px; height:auto;">

## Common Causes

- Insufficient pull compensation
- Low stitch density
- Improper underlay settings
- Fabric movement during embroidery
- Incorrect stabilizer selection
- Poor digitizing for the material being used

## How to Fix It

1. Increase the pull compensation in your embroidery software.
2. Check whether the stitch density is appropriate for the fabric.
3. Use the correct underlay type.
4. Hoop the fabric securely with the proper stabilizer.
5. Test the design before running the final production.
