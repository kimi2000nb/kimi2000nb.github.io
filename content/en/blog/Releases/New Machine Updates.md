## **System Software Update** （**[Machine Model] Upgrade** ✅） 

Includes Tutorial Videos and Update Files

Please follow the steps below to complete the update:

1. After downloading the update files, extract them to the root directory of the USB drive.
   If the three program files are **extracted** inside a folder, please move the three files directly to **the USB drive root directory** (outside the folder).
2. During the second update step, you need to choose **one of the two programs** to run the update.
   Both programs serve the same purpose, but different machines may require different programs to complete the update successfully. If one program does not work, try the other one. There is no need to worry.
